Uncrate is a minimal static documentation generator that builds sites based on the file structure of markdown files.

[![npm version](https://badge.fury.io/js/uncrate.svg)](https://badge.fury.io/js/uncrate)

## Usage

```sh
$ npx uncrate -h
```
